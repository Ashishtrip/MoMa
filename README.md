# MoMa Money Manager

## Overview

Money manager web app built with HTML, CSS, and JavaScript.

## Files

- index.html - Main HTML structure
- css/styles.css - Styling (dark theme)
- js/app.js - Application logic

## Features

- Add, edit, delete transactions
- Filter and sort by category, date, amount
- Financial summary (income, expenses, balance)
- Pie charts for category distribution
- CSV data export
- Data persistence (localStorage)
- Form validation
- Responsive design

## Usage

1. Open index.html in browser
2. Or use Live Server extension
3. Or run: python -m http.server 8000
